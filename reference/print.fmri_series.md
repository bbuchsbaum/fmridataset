# Print Method for fmri_series Objects

Display a concise summary of an fmri_series object, including
dimensions, selector type, backend, and data orientation.

## Usage

``` r
# S3 method for class 'fmri_series'
print(x, ...)
```

## Arguments

- x:

  An `fmri_series` object

- ...:

  Additional arguments (unused)

## Value

Returns `x` invisibly. Called for its side effect of printing to the
console.
