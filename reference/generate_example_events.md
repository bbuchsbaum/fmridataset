# Generate Example Event Table

Generate Example Event Table

## Usage

``` r
generate_example_events(
  n_runs = 2,
  events_per_run = 4,
  TR = 2,
  run_length = 100
)
```

## Arguments

- n_runs:

  Number of runs

- events_per_run:

  Events per run

- TR:

  Repetition time

- run_length:

  Length of each run in TRs

## Value

Data frame with event information
