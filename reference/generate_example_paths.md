# Create Example File Paths for Vignettes

Create Example File Paths for Vignettes

## Usage

``` r
generate_example_paths(
  n_runs = 2,
  subject_id = "sub-001",
  base_path = tempdir()
)
```

## Arguments

- n_runs:

  Number of runs

- subject_id:

  Subject identifier

- base_path:

  Base directory path

## Value

Character vector of file paths
