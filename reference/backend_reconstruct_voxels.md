# Reconstruct Voxel Data from Latent Backend

Reconstruct Voxel Data from Latent Backend

## Usage

``` r
backend_reconstruct_voxels(backend, rows = NULL, voxels = NULL)
```

## Arguments

- backend:

  A latent_backend object

- rows:

  Optional row indices (timepoints)

- voxels:

  Optional voxel indices

## Value

Matrix of reconstructed voxel data
